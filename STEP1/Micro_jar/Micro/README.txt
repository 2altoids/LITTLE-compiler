Submission structure:
	1: Main.java is the Driver class;
	2: CustomToken.java is additional class that must be 
	   included to the project, just in case if you want 
	   to compile on your computer;
	3: MicroGrammar.g4 is ANTLR4 grammar file for the Micro language;
	4: Micro.jar is the Java executable program;
	   4.1: Micro.jar includes Main.java, CustomToken.java and 
	        all files which were generated by ANTLR tool;
	5: We have Tokenizer.bat if you will run it, it will generate and compile all files
	   5.1: to run it type: java Main file_name.micro

How to run Micro.jar in Windows:
	1: java -jar Micro.jar file_name.micro
	2: file_name.micro must be in the same directory
